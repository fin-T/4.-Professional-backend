## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# development with swc builder
$ npm run start -- -b swc

# watch mode
$ npm run start:dev

# watch mode with swc builder
$ nest start -b swc -w

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
